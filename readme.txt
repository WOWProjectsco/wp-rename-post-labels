== Description ==

= Customizing Default Post Labels =

"Renamem Post Labels by WOWDevShop" is a clean and easy-to-use plugin that let's you rename and customize the default post labels and icon for the admin panel.

= Support =

Looking for a helping hand? [View plugin documentation](http://docs.wowdevshop.com/wordpress/rename-post-labels).

= Get Involved =

Looking to contribute code to this plugin? Go ahead and [fork the repository over at GitHub](https://github.com/wowdevshop/wp-rename-post-labels).
(submit pull requests to the latest "release-" branch)

== Usage ==

To change the icon and customize the default post labels use the following steps:

1. Go to Settings > Rename Default Post Labels
2. Write you custom Labels.
3. Save your changes.

== Installation ==
Installing "Rename Post Labels by WOWDevshop" can be done either by searching for "Rename Post Labels by WOWDevShop" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress
